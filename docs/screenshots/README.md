## Capture guidelines

- Use a real (or representative) paper + repository so the extracted values look meaningful, not empty/fallback defaults.
- Capture at a reasonable browser width (~1280px) so text and badges are legible.
- Save as PNG, optimized/compressed if possible, directly in this directory (no subfolders).
